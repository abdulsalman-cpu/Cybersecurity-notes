## 1. What does DFIR stand for in cybersecurity?

### Options
- Distributed Firewall and Intrusion Reporting  
- Data Filtering and Information Recovery  
- Dynamic Filtering and Incident Remediation  
- Device Failure and Infrastructure Repair  
- Digital Forensics and Incident Response  

### Correct Answer
Digital Forensics and Incident Response  

### Explanation
DFIR combines two important areas of cybersecurity:

- Digital Forensics: Investigating systems and collecting evidence after a cyber incident  
- Incident Response: Detecting, responding to, and recovering from cyber attacks  

Together, DFIR helps organizations understand attacks, reduce damage, and prevent future incidents.

## 2. In a comprehensive penetration test, why is directory enumeration typically performed after port scanning?

### Options
- Directory enumeration works better on older systems  
- Port scanning identifies web services that can then be enumerated  
- Port scanning must be done on weekends only  
- Directory enumeration requires root privileges  
- Directory enumeration is faster than port scanning  

### Correct Answer
Port scanning identifies web services that can then be enumerated  

### Explanation
Port scanning is performed first to discover open ports and running services. Once web services (like HTTP/HTTPS) are identified, directory enumeration is used on those services to find hidden paths and files. This makes the process efficient and targeted.

## 3. True or False: Static malware analysis involves running the malicious program in a controlled environment.

### Options
- True  
- False  

### Correct Answer
False  

### Explanation
Static malware analysis examines the malware without executing it. Running the malware in a controlled environment is part of dynamic analysis.

## 4. True or False: Ethical hackers operate with permission to find vulnerabilities before malicious attackers exploit them.

### Options
- True  
- False  

### Correct Answer
True  

### Explanation
Ethical hackers are authorized to test systems for weaknesses. Their goal is to find and fix vulnerabilities before attackers can exploit them.
